Procedure Evaluation using Language Modelling

In this paper, we developed a synthetic dataset of procedural steps extracted from student manuals using Chain-of-Thought (CoT) prompting on GPT-4, enabling structured data generation for procedural validation. To enhance error detection capabilities, I fine-tuned LLaMA-7B, Mistral-7B, and T5-13B on this dataset, training them to identify and correct erroneous procedural steps. Additionally, I conducted a comparative analysis evaluating the performance of fine-tuned models versus few-shot prompting on GPT-4, assessing their accuracy, efficiency, and generalization capabilities in procedural validation. This research contributes to AI-driven process automation, error detection, and instructional consistency across structured datasets.
Technologies: Python, Pytorch, Keras, Sklearn, LLM
Publication: In Progress
Conference: In Progress

DBWorkout: A Collaborative and Gamified Web-Based Platform for SQL Learning

 In this paper, we present DBWorkout, a web-based interactive SQL learning platform that enhances student engagement through real-time collaboration, automated feedback, and gamification. The platform features an SQL editor with instant feedback, WebSocket-based collaboration, and leaderboards to motivate learners. A cloud-based infrastructure with JWT authentication ensures security and scalability, while an analytics dashboard provides instructors with insights into student performance. By integrating hands-on practice and competitive challenges, DBWorkout modernizes SQL education, fostering industry-relevant skills in an engaging and interactive environment.
Technologies: Python, Flask, PostgreSQL, SQL, React, Javascript
